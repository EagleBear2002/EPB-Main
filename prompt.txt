使用 Java 开发一个可视化界面，命名为 epb.exe。
界面应该类似当前的 JFLAP7.1.jar，主界面美观一些。
主界面应该包括四个按钮：“图片标注（labelme.exe）”、“数据可视化（gold-data-visual.exe）”、“EPB 识别（epb-detection.exe 或 .py ）”、“数据训练（epb-train.exe）【试用功能】”
默认在 epb.exe 所在当前文件夹查找上述四个 .exe，如果当前没有，则打开文件选择对话框，供用户选择 .exe，然后打开（或者将四个 .exe 也一起打包，这样更方便，看是否可行）。
应该允许用户同时打开多个页面，每个页面都关闭后返回主菜单。
